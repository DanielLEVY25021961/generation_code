Objectif_du_Projet-generation_code.txt :

Ce projet permet LA GENERATION AUTOMATIQUE DU CODE D'UNE APPLICATION JAVA

1 - L'UTILISATEUR CREE UN NOUVEAU PROJET MAVEN SIMPLE (sans artefact) 
"projet cible" sous ECLIPSE.

2 - LE GENERATEUR DE CODE RECOPIE TOUTE L'ARBORESCENCE D'INFRASTRUCTURE 
D'UNE APPLICATION JAVA DANS LE PROJET CIBLE (couches n-tiers).

3 - LE GENERATEUR DE CODE COPIE TOUTES LES CLASSES D'INFRASTRUCTURE 
(valables pour tous les projets Java) DANS LE PROJET CIBLE.

4 - L'UTILISATEUR CREE UN CONCEPT (simplifi√© avec seulement ses attributs) 
COMME Compte, Personne, Banque, Client, ... DANS LA COUCHE METIER 
DU PROJET CIBLE.

5 - LE GENERATEUR DE CODE GENERE POUR CE CONCEPT 
L'OBJET METIER ET LES DAO, DTO, ENTITIES, CONTROLLERS, SERVICE, ...
PLUS LEURS TESTS JUNIT.
